# restaurant-gsap


A Landing page with a tiffin theme using gsap animations for a premium looking ui and subtle animations that add a nice feel to users while scroll through the website




## Authors

- [@varshith-jureddi](https://github.com/varshith-jureddi)


## Features

- Eye catching Hero section
- Subtle effects all over the site
- Smooth experience from start to end


## Tech Stack
[![My Skills](https://skillicons.dev/icons?i=react,tailwind,javascript,&perline=4)](https://skillicons.dev)

React, TailwindCSS, GSAP, JavaScript



## Run Locally

Clone the project

```bash
  git clone https://github.com/varshith-jureddi/restaurant-gsap
```

Go to the project directory

```bash
  cd restaurant-gsap
```

Install dependencies

```bash
  npm install
```

Start the Frontend

```bash
  npm run dev
```
